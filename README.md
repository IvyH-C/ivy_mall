# ivy_mall
个人能力展示项目


## 项目介绍

​	ivy_mall电商系统是一个作品展示项目。由于之前与公司签订了保密协议的关系，协议不允许把公司项目的源代码上传至个人Github或以其他形式对外展示。因此开发ivy_mall电商系统，用于向用人单位展示个人的编码风格、业务设计水平以及技术应用水平。
